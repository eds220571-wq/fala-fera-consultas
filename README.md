# Fala Fera Consultas

Dashboard web em HTML, CSS e JavaScript puro para consultas usando BrasilAPI.

## Recursos da V2

- Consulta CEP
- Consulta CNPJ
- Consulta DDD
- Consulta Feriados nacionais
- Consulta Bancos
- Leitor básico PIX copia e cola
- Histórico local no navegador
- Tema claro/escuro
- Layout responsivo
- Pronto para GitHub Pages

## Como publicar no GitHub Pages

1. Envie estes arquivos para o repositório `fala-fera-consultas`.
2. Entre em **Settings > Pages**.
3. Em **Build and deployment**, selecione **Deploy from a branch**.
4. Escolha a branch `main` e a pasta `/root`.
5. Salve.

O site ficará disponível em:

```txt
https://eds220571-wq.github.io/fala-fera-consultas/
```
